# Data Structures and Algorithms 1

This repository contains the source code and projects developed during the **Data Structures and Algorithms 1** course of the Computational Mathematics undergraduate program at UNIFESP.

## Topics and Implementations
The files in this repository demonstrate the practical application of fundamental and advanced data structures using C:
* **Dynamic Memory & Structs:** Array manipulation and dynamic allocation (`ex1`).
* **Stacks:** Implementation using linked lists, including a Polish calculator simulator (`ex2`).
* **Queues:** Dynamic queue implementation for a ticketing/messaging system (`ex3`).
* **Complex Structures (List of Stacks):** Managing multiple stacks dynamically with operations like split and merge (`ex4`).
* **Doubly Circular Linked Lists:** Bi-directional traversal and element management (`ex5`).
* **Sparse Matrices:** Mathematical operations (addition, subtraction, multiplication) optimized through linked lists (`ex6`).
* **Binary Trees:** Reconstructing binary trees from Pre-order and In-order traversals to generate Post-order traversals (`ex7`).

## Technologies Used
* C Language
* GCC Compiler
* Memory management (`malloc`, `free`)
